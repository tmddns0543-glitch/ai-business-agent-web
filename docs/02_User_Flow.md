# AI Business Agent Web User Flow

## 1. 신규 사용자 흐름

서비스 소개
→ 회원가입
→ 이메일 인증
→ 사업장 생성
→ 업종 선택
→ 사용 매출채널 선택
→ 채널별 수수료 확인
→ 첫 매출 입력
→ 오늘 예상 정산금액 확인
→ 홈 진입

## 2. 로그인 사용자 일일 흐름

로그인
→ 홈
→ 오늘 정산 상태 확인
→ 매출 입력
→ 예상 수수료 및 정산금 확인
→ 비용 입력
→ 홈 또는 경영성과 확인

## 3. 가이드 매출입력 흐름

정산 날짜 확인
→ 활성 매출채널 확인
→ 채널별 매출 입력
→ 주문 수 입력
→ 입력 요약
→ 예상 수수료 확인
→ 예상 정산금액 확인
→ 저장
→ 완료 화면

## 4. 빠른 매출입력 흐름

정산 날짜 확인
→ 활성 채널 전체 입력
→ 요약 확인
→ 저장
→ 완료 화면

## 5. 비용입력 흐름

날짜 확인
→ 발생한 비용 유형 선택
→ 선택한 비용만 입력
→ 거래처 및 메모 입력
→ 요약 확인
→ 저장

## 6. 경영성과 흐름

경영성과
→ 조회 월 선택
→ 매출 확인
→ 수수료 및 예상 정산금 확인
→ 비용 구조 확인
→ 영업이익 확인
→ 세부 항목 확인

## 7. 피드백 흐름

어느 화면에서든 피드백 버튼
→ 버그 / 불편 / 기능제안 선택
→ 내용 입력
→ 현재 페이지 및 사용자 정보 자동 첨부
→ 제출
