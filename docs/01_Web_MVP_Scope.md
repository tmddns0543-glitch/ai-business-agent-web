# AI Business Agent Web MVP Scope

## 1. 제품 목표

AI Business Agent Web은 외식업 점주가 매일 짧은 시간 동안 매출과 비용을 입력하고,
플랫폼 수수료, 예상 정산금액, 손익 구조를 쉽게 확인할 수 있도록 돕는 웹서비스다.

초기 목표는 AI 기능이 아니라 다음 문제를 해결하는 것이다.

- 플랫폼별 매출을 한곳에서 입력
- 플랫폼별 예상 수수료 자동 계산
- 실제 예상 정산금액 확인
- 일별·월별 매출과 비용 관리
- 월별 경영성과 확인
- 배달대행사 예수금 흐름 확인

## 2. 초기 베타 대상

- 대한민국 외식업 점주
- 배달 플랫폼을 1개 이상 사용하는 사업장
- 매출과 비용을 직접 관리하고 싶은 사용자
- 비공개 베타 10명부터 시작

## 3. Web MVP 포함 기능

### 비회원 영역

- 서비스 소개
- 로그인
- 회원가입
- 비밀번호 재설정
- 베타 이용 안내

### 회원 영역

- 사업장 초기 설정
- 홈
- 매출 입력
- 비용 입력
- 배달대행사 현황
- 경영성과
- 설정
- 피드백 제출
- 로그아웃

### 매출 입력

- 가이드 입력
- 빠른 입력
- 플랫폼별 매출
- 플랫폼별 주문 수
- 예상 플랫폼 수수료
- 예상 정산금액
- 기존 입력 수정

### 비용 입력

- 원재료비
- 포장재비
- 인건비
- 시설·공과 비용
- 플랫폼·배달 비용
- 기타 운영비
- 사용자 정의 비용항목
- 기존 입력 수정

### 경영성과

- 월 선택
- 총매출
- 예상 수수료
- 예상 정산금액
- 영업비용
- 영업이익
- 영업이익률
- 비용 구조
- 플랫폼별 매출 비중

### 설정

- 사업장 정보
- 사용 매출채널
- 채널별 수수료 설정
- 비용항목 설정
- 기초·기말재고 설정

## 5. 핵심 UX 원칙

- 모바일 화면을 우선 설계한다.
- 처음 사용하는 사람도 설명 없이 입력할 수 있어야 한다.
- 한 화면에 너무 많은 정보를 넣지 않는다.
- 사용하지 않는 채널과 비용항목은 노출하지 않는다.
- 입력 직후 사용자가 얻는 결과를 바로 보여준다.
- 금액은 항상 천 단위 구분기호를 표시한다.
- 저장 전에 입력 내용을 검토할 수 있어야 한다.
- 기존 입력 수정과 취소가 쉬워야 한다.

## 6. 성공 기준

- 회원가입 후 첫 매출 입력 완료율 70% 이상
- 첫 사용자가 10분 안에 첫 정산 완료
- 베타 사용자의 주 3회 이상 사용률 50% 이상
- 4주 후 지속 사용률 50% 이상
- 핵심 손익 계산 오차 2% 이내
