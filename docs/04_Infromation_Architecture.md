## 2. 회원 화면

### 홈

- 오늘 매출 입력 상태
- 오늘 비용 입력 상태
- 오늘 정산 시작
- 이번 달 총매출
- 예상 플랫폼 수수료
- 예상 정산금액
- 월말 예상 영업이익
- 최근 7일 매출
- 플랫폼별 매출 비중

### 매출입력

- 가이드 입력
- 빠른 입력
- 기존 입력 조회
- 입력 수정
- 입력 완료 결과

### 비용입력

- 발생 비용 선택
- 비용 금액 입력
- 거래처
- 메모
- 기존 입력 조회
- 입력 수정

### 경영성과

- 월 선택
- 총매출
- 예상 수수료
- 예상 정산금액
- 영업비용
- 영업이익
- 영업이익률
- 비용 구조
- 플랫폼별 매출 비중

### 더보기

- 배달대행사 현황
- 설정
- 피드백
- 로그아웃

## 3. 설정
