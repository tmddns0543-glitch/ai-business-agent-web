# AI Business Agent Web Mobile Wireframe

## 1. 문서 목적

이 문서는 AI Business Agent Web의 모바일 화면 구조를 정의한다.

모든 화면은 모바일을 기준으로 설계한다.

PC 화면은 모바일 레이아웃을 확장하는 방식으로 개발한다.

---

# 2. 설계 원칙

모든 화면은 다음 원칙을 따른다.

- 한 화면에는 하나의 핵심 행동만 제공한다.
- 중요한 버튼은 화면 하단에서 한 손으로 누를 수 있어야 한다.
- 숫자는 크게 보여준다.
- 글보다 행동이 먼저 보이도록 한다.
- 사용자가 생각하지 않아도 다음 행동을 알 수 있어야 한다.
