# AI Business Agent Web User Journey

## 1. 문서 목적

이 문서는 사용자가 서비스를 처음 접하는 순간부터
매일 반복적으로 사용하는 흐름을 정의한다.

모든 UX는
이 User Journey를 기준으로 설계한다.

"생각보다 쉽다."

를 느끼게 하는 것이다.
